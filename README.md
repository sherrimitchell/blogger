# Blogging Site
This a basic blogging application built using Rails. The application allows the user to:
* Sign up and sign in
* Write posts
* Edit and delete posts
* Tag posts
* Add comments
* Admin ability to delete comments
* View posts and comments ordered by how recent they are
* Click on a post and visit a page

**Languages, Libraries, and Frameworks:** Ruby on Rails, Paperclip

**Databases:** SQLite (Test, Development)

**Development Tools, Gems and Methodologies:**

Devise for user authentication
